# AdjustTimeBon

## これは何？
BonDriverを使用して放送波のTOT（Time Offset Table）をキャプチャし、時計合わせを行うコンソールアプリケーションです。  
（タスクスケジューラで実行できるように、TVTestの「TOTで時刻合わせ」相当の機能を抜き出してコンソールアプリケーション化したもの。）


DBCTRADOさんのLibISDBを利用・参照させて頂きました。 ありがとうございます。

## 使用方法
以下を同一ディレクトリに配置すれば実行できるはずです。
* AdjustTimeBon.exe  
* AdjustTimeBon.ini  
* BonDriverのDLLファイル  

AdjustTimeBon.iniで各種設定を変更することができます。
